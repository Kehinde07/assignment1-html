Html Formatting elements

Bold text: This is a bold text. This Html element defines a bold text. For example, Please subscribe to my youtube channel it is known as sisi-kenny.

Important text: This is a strong text. This Html element places the importance of a particular text. For example, Please subscribe to my youtube channel it is known as sisi-kenny.

Italic text: This text is an italic text. This Html element defines a part of a text in form of a phrase or like voice. For example, My content is all about life-style.

Emphasized text: This is an emphasized test. This Html element is used to emphasize a particular text. For example, My content is all about life style and beauty as at now.

Marked text: This is a marked text. This Html element defines a part of a text that should be marked or highlighted. For example, My content was all about life style and beauty but I introduced accounting contents last week.

Small text: This is a small text. This Html element defines smaller text. For example, I am getting little or none engagement in my accounting content compared to my other contents which seem to be doing much more better.

Deleted text: This is a deleted text. This Html element defines a text that has been deleted. For example, I removed Accounting from my content.

Inserted text: This is an inserted text. This Html element defines a text that has been inserted in a document. For example, I switched up my content and added programming to my content.

Subscript: This is a subscripted text. This element is a subscript text which appears below the normal line. For example, As at today, I believe that by the end of the 3rd of the year my channel will be the most popular channel on youtube.

Superscript: This is a superscripted text. This Html element is a superscript text which appears above the normal line. For example, As at today, July 14th my subscribers have increased by 15 million.
